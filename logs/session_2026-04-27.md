### Phase 3：Bug 修正（Commits: `3160d03` → `d55d566` → `cebe00e`）

使用者在 Windows PyCharm 環境執行時遇到連續錯誤：

| 錯誤訊息 | 根因 | 修正方式 |
|---|---|---|
| `ImportError: cannot import name 'QFileSystemModel'` | PyQt6 新版移除此 class | 改用 `QTreeWidget` 手動掃描目錄 |

## 重要技術決策記錄

| 決策 | 選擇 | 原因 |
|---|---|---|
| 檔案樹元件 | `QTreeWidget`（手動掃描） | PyQt6 新版移除 `QFileSystemModel` |
| 型別標注相容 | `from __future__ import annotations` | 支援 Python 3.8/3.9 |
| 批量處理 | `ProcessPoolExecutor`（非 `ThreadPoolExecutor`） | CPU-bound 影像處理，多進程才有效加速 |
| GL 偵測 | `cv2.inRange(gl_min, gl_max)` | 半導體 SEM 影像中 MG 亮度有上下界，單向閾值會誤包含過亮區域 |
| Annotated 標籤 | 純數字，無背景框 | 使用者要求簡潔，顏色本身已代表語意 |
| Overlay 控制位置 | Viewer header（Annotated 時才顯示） | 情境式顯示，不佔用常設面板空間 |
