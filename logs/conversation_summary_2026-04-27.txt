【第一階段】需求釐清（多輪問答）
────────────────────────────────────────────────────────────────────────────────

使用者提出要開發一套 SEM 影像自動量測工具，AI 透過多輪 Q&A 釐清需求：

Q1：主要量測哪一層？
A：MG（Metal Gate）層，垂直亮條紋，目標量測 CMG Y-CD。

Q2：CMG 是什麼？量什麼？
A：Cut Metal Gate，水平方向切割，將 MG 切成上下兩段。
   Y-CD = 上段 MG 底部 → 下段 MG 頂部的垂直距離（像素 × nm/pixel）。

Q3：每張影像有幾個 CMG cut？
A：可能有多個 CMG cut，每個 cut 有多根 MG column，每一對都需量測。

Q4：輸出方式？
A：每個 CMG cut 輸出所有 MG column 的 Y-CD 值。

Q5：如何偵測 MG？
A：GL 閾值（後來改為範圍），即時預覽滑桿。

Q6：批量處理規模？
A：1000+ 張影像，需多核心平行處理。

Q7：輸出格式？
A：CSV、Excel（含統計頁）、JSON、HTML 報告、標注影像。

Q8：GUI 框架？
A：PyQt6 / PySide6，具備 Zoom/Pan 影像預覽。

Q9：結果呈現方式？
A：影像 + 對應表格（下方），MIN 紅、MAX 藍。

Q10：KLARF 整合？
A：Phase 2，本次先不做。

Q11：介面語言？
A：英文。

Q12：作業系統？
A：Windows 10/11。

Q13：GL 閾值調整？
A：即時預覽 Slider（後來改為 Min/Max 雙滑桿）。

【第三階段】初始建構（全部從零寫起）
────────────────────────────────────────────────────────────────────────────────

依序建立以下檔案：

[Core 模組]
- image_loader.py：多格式載入 → uint8 灰階
- preprocessor.py：前處理 pipeline（Gaussian → CLAHE → 閾值 → 形態學）
- mg_detector.py：連通元件 → Blob 列表 + 面積過濾
- cmg_analyzer.py：CMG Y-CD 主演算法
  * Union-Find 做 X-range 分組
  * 同欄 blob 排 Y → 相鄰對 = CMG gap
  * 按 mid_y 聚類跨欄 gap → 同一 CMG cut 事件
  * 每個 cut 標記 MIN / MAX
- annotator.py：在影像上繪製量測標注

[GUI 模組]
- image_viewer.py：QGraphicsView，滾輪縮放、拖曳平移
- file_tree_panel.py：QTreeView + QFileSystemModel（初版）
- control_panel.py：nm/pixel 輸入 + threshold slider + morph 參數
- results_panel.py：QTableWidget 結果表格
- batch_dialog.py：QDialog + QThread + ProcessPoolExecutor
- main_window.py：主視窗布局

[Output 模組]
- csv_exporter.py
- excel_exporter.py（含 Statistics 頁 + 紅藍標色）
- json_exporter.py
- report_generator.py（HTML + matplotlib 直方圖）

[測試]
- test_cmg_analyzer.py：10 項單元測試（全部通過）

[Entry Point]
- main.py：freeze_support + if __name__ == '__main__' 守衛

【第四階段】Bug 修正（使用者在 Windows PyCharm 執行時連續報錯）
────────────────────────────────────────────────────────────────────────────────

Bug 1：ImportError: cannot import name 'QFileSystemModel' from 'PyQt6.QtWidgets'
  原因：PyQt6 新版已移除 QFileSystemModel
  修正：file_tree_panel.py 改用 QTreeWidget + 手動遞迴掃描目錄
  影響檔案：src/gui/file_tree_panel.py

Bug 2：TypeError: unsupported operand type(s) for |: 'type' and 'type'
  原因：Python 3.9 以下不支援 X | Y 型別語法（Python 3.10+ 才有）
  修正：在所有缺少的 .py 檔頂端加入 from __future__ import annotations
  影響檔案：image_loader.py、mg_detector.py、preprocessor.py、
            _common.py、csv_exporter.py、excel_exporter.py、json_exporter.py

Bug 3：AttributeError: 'RenderHint' object has no attribute 'RenderHint'
  原因：self.renderHints() 回傳 flags 物件而非 enum 類別，
        寫法 self.renderHints().RenderHint.Antialiasing 無效
  修正：直接刪除該行（預設已關閉反鋸齒，此行本為多餘）
  影響檔案：src/gui/image_viewer.py
