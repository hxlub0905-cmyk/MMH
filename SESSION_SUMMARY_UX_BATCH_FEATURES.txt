── 問題 1：Measure 按下「Save Rcp」後，Batch / Recipe 清單不即時更新 ─────────

  根本原因：
    MeasureWorkspace 儲存 Recipe 後只 emit recipe_saved signal，
    但 BatchWorkspace 的 recipe combo 沒有訂閱。

  修法：
    WorkspaceHost._connect_signals() 新增：
      self._measure.recipe_saved → self._batch.refresh_recipe_selector
      self._measure.recipe_saved → lambda: self._recipe.refresh_from_registry()
    讓三個 workspace 的 recipe selector 保持同步。

── 問題 2：確認 Batch Worker 是否真正 Multi-Process ──────────────────────────

  確認結果：
    MeasurementEngine.run_batch() 使用 ProcessPoolExecutor，
    worker function (_worker_run_image) 為 top-level picklable function，
    確實跑在獨立子程序，非 thread-based。屬告知性問題，無需修改。

── 問題 3：Report CD 統計新增 Q25 / Q75 ──────────────────────────────────────

  使用者確認 Q50（Median）已存在，只需補 Q25 / Q75。

  修法：
    report_workspace.py _refresh_stats() 中，使用
    statistics.quantiles(ok_vals, n=4) 取 qs[0]（Q25）與 qs[2]（Q75），
    插入 Median 之後顯示。
    report_generator.py _compute_stats() 同步新增 Q25 / Q75 欄位。

── 問題 4：CSV / Excel / JSON / HTML / Images 5 個 Export 按鈕整合成一個 ─────

  修法：
    新增 _ExportDialog(QDialog) — 含：
      · Output folder 選擇列
      · 格式 checkbox：CSV、Excel（兩者若無 pandas 則灰底警告）、
        JSON、HTML Report、Overlay Images
    report_workspace.py 改為單一「Export…」按鈕，點擊後開 dialog。
    匯出錯誤逐項收集後統一彈窗顯示。

── 問題 5：跑完 Batch → Review tab UI 凍結 ───────────────────────────────────

  根本原因：
    ReviewWorkspace.load_batch_run() 在 main thread 一次性重建所有
    MeasurementRecord，資料量大時阻塞 event loop。

  修法：
    load_batch_run() / report_workspace.py load_batch_run() 中，
    每處理 200 筆呼叫一次 QApplication.processEvents()，
    讓 UI 保持回應。

── 問題 6：Batch Image Results 改為即時 Log 顯示 ─────────────────────────────

  根本原因：
    舊版在 _on_single_finished() 後才一次將結果填入 QListWidget。

  修法：
    _on_progress(done, total, name, status) slot 中，
    每筆完成時立即 append 一個 QListWidgetItem，
    並以顏色區分 OK（darkGreen）/ FAIL（red）。

── 問題 7：Export Images 改為只輸出 Overlay，不輸出 Raw ─────────────────────

  修法：
    新增 _export_overlays_to() 方法：
      · 讀取原始影像 → draw_overlays() → 僅存 annotated PNG
      · 帶有 QProgressDialog（可取消）
    移除原本同時輸出 raw 的邏輯。

── 問題 8：無 pandas 時 Export 跳出 "No module named 'pandas'" ──────────────

  修法：
    _ExportDialog 中呼叫 _check_pandas()，
    若 pandas 未安裝則 CSV / Excel checkbox 顯示警告文字並 setEnabled(False)，
    不再丟出未處理例外。

Feature 1：邊界 Blob 濾除（Border Blob Filter）
────────────────────────────────────────────────────────────────────────────────

  動機：
    Blob bbox 若壓在影像邊界，量測到的 CD 會極度不合理（最小值 outlier），
    影響 Batch 統計。

  Schema 擴充：
    edge_locator_config["border_margin_px"]（預設 0，表示關閉）

  修改檔案：

  src/core/recipes/cmg_recipe.py
    · detect_features() 幾何 filter 之後插入：
        _border_px = int(self._descriptor.edge_locator_config.get("border_margin_px", 0))
        if _border_px > 0:
            _h, _w = mask_roi.shape[:2]
            blobs = [b for b in blobs
                     if b.x0 >= _border_px and b.y0 >= _border_px
                     and b.x1 <= _w - _border_px and b.y1 <= _h - _border_px]
    · _card_to_descriptor() edge_locator_config dict 加入
        "border_margin_px": int(card.get("border_margin_px", 0))

  src/gui/workspaces/recipe_workspace.py
    · Analysis tab 新增 self._border_margin = QSpinBox(0-200)
    · _load_descriptor_to_form() 讀取 ec.get("border_margin_px", 0)
    · _save_recipe() 寫回 "border_margin_px"

  src/gui/workspaces/measure_workspace.py
    · _build_edge_locator_panel() 新增 self._ec_border QSpinBox
    · _on_recipe_combo_changed() 同步讀取
    · _run_with_recipe() patch descriptor 時帶入
    · _analyze_with_cards() 鏡像同一濾除邏輯

── Bug 1：Batch Error: 'recipe_ids' ─────────────────────────────────────────

  症狀：多組 Batch 執行時跳出 "Batch Error: 'recipe_ids'"。

  根本原因：
    batch_workspace.py _run_batch() 組建 valid_rows 時，
    使用 "recipe_id": recipe_id（字串），
    但 measurement_engine.py run_multi_batch() 讀取的 key 為
    "recipe_ids"（list of str）→ KeyError。

  修法：
    valid_rows.append() 改為 "recipe_ids": [recipe_id]
    單組路徑的 _BatchWorker 建構也改用 r["recipe_ids"]。

── Bug 2：Dataset 設定框 Y 方向太窄 ─────────────────────────────────────────

  症狀：Dataset scroll area 僅 200px，新增多列時擁擠難以操作。

  修法：
    scroll.setMinimumHeight(120)
    scroll.setMaximumHeight(340)
    （原本為固定 maxHeight=200px）
