二、KLARF Overlay 座標問題（根本原因）
───────────────────────────────────
【問題現象】Overlay 完全看不到，推測跑到圖外

【初次錯誤診斷】
誤以為是 16-bit TIFF 未正規化導致顏色近黑。
修正後（加 normalize）仍看不到 overlay。

【正確根本原因】
XREL/YREL 是 KLARF die-corner 絕對座標（單位 nm，通常數百萬量級），
不是影像中心的相對偏移量。

錯誤做法：px = cx + xrel_orig / nm_per_pixel
          → 結果：影像中心 + 數百萬像素 → 遠在畫面外

正確做法：
  Orig 位置 = 影像中心（SEM 影像固定以量測點為中心）
  New 位置  = 中心 + 位移
  dx_px = (xrel_new - xrel_orig) / nm_px
  dy_px = (yrel_orig - yrel_new) / nm_px   ← Y 軸反向
  orig_pt = (W//2, H//2)
  new_pt  = (W//2 + dx_px, H//2 + dy_px)

修復後 overlay 正確顯示。
